# 👗 MAASH AI Studio

## 💡 About
MAASH AI Studio is a web application that helps users find outfit ideas for different occasions such as casual, party, and wedding.

Many people find it difficult to decide what to wear for events. This app solves that problem by providing instant outfit suggestions along with visual design ideas.

---

## 🌐 Live App
https://anammehboob878-eng.github.io/MAASH-ai-studio/

---

## ✨ Features
- AI-based outfit suggestion system  
- Categories: Casual, Party, Wedding  
- Displays 5 outfit designs for each category  
- Interactive buttons for quick selection  
- Simple and user-friendly interface  

---

## 🤖 AI Feature
The app includes a simple AI system based on user input.

It reads keywords like:
- wedding  
- party  
- casual  

Based on the keyword, it:
1. Generates outfit suggestions  
2. Shows related outfit images  

**Example:**  
If the user types "wedding", the app suggests formal/traditional outfits and displays wedding designs.
